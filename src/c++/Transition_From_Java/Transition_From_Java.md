# Домашнее Задание "Переход из Java в C++"

Для начала стоит понимать, что Java и C++ - очень похожие языки.
Синтаксис Java был частично позаимствован у C++, особенно заметно проглядывается
в таких основополагающих вещах как численные типы (`int`, `long`, `float`, `double`), условные операторы (`if`, `else`) и циклы (`while`, `for`, `do ... while`).

Например вот этот код на Java будет работать в C++ лишь с минимальными изменениями:

###Java Version
```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int a, b;

        System.out.printf("Enter first number: ");
        a = sc.nextInt();

        System.out.printf("Enter second number: ");
        b = sc.nextInt();

        int sum = a + b;

        System.out.printf("Sum = %d\n", sum);
    }
}
```
### C++ Version
```c++
#include <cstdio>

int main() {
    int a, b;

    printf("Enter first number: ");
    scanf("%d", &a);

    printf("Enter second number: ");
    scanf("%d", &b);

    int sum = a + b;

    printf("Sum = %d\n", sum);

    return 0;
}
```

Как видно, отличия не велики, но имеются. Давайте рассмотрим их.

## Различия между языками

### 1. Парадигма Object Oriented Programming

В Java мы писали все наши коды внутри классов (например `class Main`),
а начальной точкой запуска программы был метод `main` (e.g. `public static void main`)
В C++ нет необходимости писать код в классах. C++ поддерживает как классы, так и отдельные функции,
не привязанные к какому либо классу.
Входной точкой C++ программы является функция `main` (e.g. `int main()`, или `int main(int argc, char** argv)` при желании).
На первых занятиях новичкам-джавистам приходится абстрагироваться от этой парадигмы, чтобы изучать основы,
потому что основы программирования не привязанны ни к каким парадигмам, как и язык C++ (хотя в больших компаниях отдают предпочтения OOP, так что это полезно знать). В C++ же есть просто начальная функция: бери и пиши.


### 2. Стандартная Библиотека и методы работы с ней

Языки Java и C++ имеют схожий синтаксис, но есть много различий, особенно когда речь идет о стандартной библиотеке и ее методах.
Это очень общирная тема, поэтому здесь только будет упомянуто различия, актуальные для примера выше.

#### 2.1 Outputing text to Standart Output (он же `stdout`)

В Java самый популярный метод - `System.out.println`, а в C++ используется специальные операторы is библиотеки `<iostream>`, такие как `std::cout` и оператор `<<`.
Это довольно хитрая штуковина, которая сильно разнится от стандартных методов вывода в других языках, но в будущем мы поймем, на чем это основано.
На данном же этапе рекомендую использовать стандартную функцию из языка C (Классический Си "без плюсов"), такую как `printf`.
Она по способу применения практически идентична методу `System.out.printf` из Java, которым вы умеете пользоваться и сможете пользоваться на первых порах.

#### 2.2 Указатели (Pointers)

В Джаве не было необходимости задаваться вопросами о том где хранится переменная на самом деле.
Все передавались из переменной в переменную или функцию по ссылке, копируя лишь адреса или примитивные данные.
В C++ же мы будем оперировать как значениями чисел, так и их адресами.
Это не так легко описать в одном предложении, но на практике станет понятно.
В данном примере принцип указателей применен в строчке `scanf("%d", &a);`, где амперсанд - `&` это оператор взятия адреса у переменной.
Переменная `a` - локальная для функции `main`, но мы хотим, чтобы в ячейку памяти переменной `a` было положено
новое значение. Запись происходит внутри функции `scanf`, которая не имеет доступа к локальной памяти функции `main`,
поэтому для записи мы передаем ей в качестве аргумента адрес переменной `a`, который принято называть указателем.
Переменная `a` имеет тип данных `int`, а выражение `&a` будет иметь тип `int *`, (a.k. int pointer) то есть указатель на `int`.

#### 2.3 Scanning inputs from Standard Input (`stdin`)

В Java мы использовали Scanner, а в C/C++ есть немало разных способов и нет приближенных аналогов как с `printf`.
В C++ используется `std::cin` для считывания текста и сохранения его в переменную, при чем преобразовывая текст в число или другой тип. В этом же примере показана функция `scanf`, которая непохожа на `Scanner.nextInt()` из Java.
