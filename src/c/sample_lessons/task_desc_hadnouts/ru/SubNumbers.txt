Напишите функцию

```

int count_subnumber_occurences(int number, int subnumber);

```

которая найдет количество содержащихся подпоследовательностей цифр `subnumber` встречающихся в данном числе `number`.
Например, число 123412001 содержит подпоследовательность 12 два раза.
2222 содержит 22 три раза:
    (22)22
    2(22)2
    22(22)

Подсказки:
    Используйте операции смещения вправо в десятичной системе - деления на степени 10.
    Используйте операции взятия остатка при делении на степень 10.
    Используйте циклы
    Поймите количество цифр в искомом и данном числе.
