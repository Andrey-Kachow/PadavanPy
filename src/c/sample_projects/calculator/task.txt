Реализовать простой калькулятор с возможностью выполнять 5 простых арифметических операции.
А именно: Сложение "+", Вычитание "-", Умножение "-", Деление "/" и Возвежение в степень "pow"

Программа должна поддерживать два режима:
    1) Интерактивная оболочка:
        - Пользователь может вбивать команды в момент работы программы.
        Программа будет ожидать ввода пользователя. (Например с помошью scanf).
        Затем выдавать ответ
    
    2) Утилита командной строки:
        - Пользователь запускает программу с набором аргументов, которые содержат в себе
        Символ операции (+, -, *, /, pow) и операнды (слагаемые, множители, и т.п.).
        Выводом программы будет одно единственное число - ответ.

Условия:
    1) Использовать int для чисел. Поддержку float и double добавлять не нужно.
    
    2) Допустимые наборы аргументов следующие (допустим исполняемый файл называется YourProgram.exe):

        Интерактивный Режим (без доп. аргументов)
        .\YourProgram.exe

        Режим утилиты командной строки
        .\YourProgram.exe --calculate <int> (+|-|*|/|pow) <int>

            где <int> - числа в десятичной системе счисления
            (+|-|*|/|pow) - операции "+", "-", "*", "/", "pow" для вычислений.

    3) В случае неправильного количества аргументов, неправильной записи опции --calculate, неправильной операции
    Вывадавать слово Error

    4) В Интерактивном режиме после каждого вычисления спрашивать у пользователя yes или no.
    В случае yes запрашиваем ввод для следующего вычисления и так далее.
    В случае no завершаем Программу.

Разрешенные допущения:

1) Не нужно проверять правильность записи чисел. Допустим пользователь вводит все правильно.
2) В случае включения пользователем Интерактивного режима не нужно делать какую бы то ни было
обработку ошибки. Допустим, что все вводы будут правильными. Лишний вывод программы может привести
к ошибке автоматических тестов (см. ниже).

PS > .\YourProgram.exe --calculate 1 + 10

PS > .\YourProgram.exe --calculate 4 pow 4

Проверка корректности решения:

    1) Скачайте Питон с сайта python.org. При установке поставьте галочку там, где говорится
    что-то про добавление python в PATH. Важно

    2) Убедитесь, что питон работает, вбив в PowerShell команду "python --version"

        Если команда python не распознана, значит нужно открыть PowerShell заново и попробовать вбить команду еше раз.
        Если и это не помогло, вы забыли поставить важную галочку. Запустите установщик Питона еще раз и почините

    3) Теперь когда Питон работает, переместите файл test_calc.py в одну и ту же папку с исполняемым файлом программы калькулятор.

    4) Добустим ваша прогамма (экзешник) называется ABCDEFG.exe

    5) Запустите тест. Вбейте команду:

            python test_calc.py .\ABCDEFG.exe
