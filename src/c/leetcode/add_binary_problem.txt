Было решено дать помощь с выделением динамической памяти,
так как выеление массива на стэке привычным образом в такой
задаче не подойдет.

Делаем вид, что у нас есть массив result нужного размера и 
реализовываем алгоритм.


```

char * addBinary(char * a, char * b){

    int length_a = strlen(a)
    int length_b = strlen(b)

    // Находим количество цифр (нулей или единиц) в ответе
    int length_result = max(length_a, length_b)
    if (a[0] = '1' && b[0] == '1') {
        length_result++;
    }

    // незабываем добавить одно лишнее место для '\0' в конце строки
    length_result += 1;

    // Выделяем необходимое количество байт для строки ответа.
    //
    // Примечание: Нельзя использовать
    // char result[99];
    // Так как размер необходимой памяти не известен.
    //
    char *result = malloc(length_result * sizeof(char));


      ///////////////////////////////////////
     ///     Здесь пишем код решения     ///
    ///////////////////////////////////////

    return result;
}
```
